# Trading Pattern - 台指期特徵分析工具

一個 MCP (Model Context Protocol) Server，用於自動化分析台指期(TXF)的交易形態特徵。

## 功能概述

- **Tick 資料處理**：自動讀取並清理台指期歷史 tick 數據
- **多時間框架**：支援 1min、5min、15min 等多個時間框架的 K 棒生成
- **特徵識別**：自動識別深坑K、高點K、低點K 等交易形態特徵
- **多框架共振**：分析不同時間框架特徵的共振信號
- **回測驗證**：量化特徵的預測力與統計勝率
- **AI 整合**：透過 MCP 接口提供給 Claude 及其他 LLM 進行智能分析

## 核心用途

把人工看盤的「形態識別」過程自動化，讓 AI 助手可以基於結構化特徵序列進行跨時間框架分析，輔助交易決策。

## 技術棧

- **語言**：Python 3.10+
- **資料處理**：pandas、numpy
- **部署**：MCP Server
- **套件管理**：uv 或 poetry

## 項目結構

```
├── data/           # 資料層（加載、清理、分類）
├── klines/         # K棒層（時間框架轉換）
├── features/       # 特徵層（形態識別演算法）
├── validation/     # 驗證層（回測、共振分析）
└── mcp_server/     # MCP 介面（與 Claude 整合）
```
